# KOBIS Movie Search System

## 프로젝트 소개
KOBIS 영화정보 데이터를 활용한 영화 검색 시스템

## 기술 스택
- Python
- Flask
- MySQL
- PyMySQL

## 주요 기능
- 영화명 검색
- 감독명 검색
- 제작연도 검색
- 페이지네이션
- 정렬
- 인덱스 성능 비교

## 실행 순서

01_create_table.py

02_insert_movie_info.py

03_create_normalized_tables.py

04_normalize_movie_data.py

05_create_index.py

python app.py
